## Getting Started

### Prerequisites

* Node.js 18+
* PostgreSQL database
* pnpm package manager

### Installation

```
# Install dependencies
pnpm install

# Run database migrations
pnpm db:push

# Start development server
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) to get started.

## Tech Stack
* Framework: Next.js 16 with React 19
* UI: shadcn/ui with Tailwind CSS
* State Management: Jotai
* Database: PostgreSQL with Prisma ORM
* Authentication: Better Auth
* Code Editor: Monaco Editor
* Type Checking: TypeScript

## About Coding platform
This project is built on top of Workflow DevKit, a powerful workflow execution engine that enables:

* Native TypeScript workflow definitions with "use workflow" directive
* Type-safe workflow execution
* Automatic code generation from visual workflows
* Built-in logging and error handling
* Serverless deployment support
Learn more about Workflow DevKit at [localhost:3000](https://localhost:3000)
